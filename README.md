# edu-Console-CS
C#コンソール課題の管理リポジトリです。
受講者単位でブランチを切って、プッシュしてください。
→詳しくは課題資料添付の手順書を参照
以下、課題を取り組むにあたって実装についての注意点です。

***

## Task1_birthdayCalculation

課題1のプロジェクトです。

`BirthdayCalculation.cs`

コンソールから実行するメインクラスとして実装してください。

`CalcLogic.cs`

計算ロジックを記述するクラスとして実装してください。
<br>
<br>
<br>
上記以外でも必要であれば適時クラスやパッケージを増やしても構いません。
